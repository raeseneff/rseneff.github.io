Definitions

How do you define a table?
To define a table you use table in < >. It creates a a table that can be used to put in your information
When do you use tr, th, td, thead, tfoot, caption?
tr: used when starting a row on your table.
th: used when you are putting in the categories for the table
td: used to put in your data for the table
thead: used when you are grouping headers in a table
tfoot: used when you are grouping footers in a table
caption: used when adding a title or caption to a table
What is the difference between colspan and rowspan?
Colspan combines cells across the table horizonatlly. Rowspan combines the cells vertically
What is the difference between cells spacing and padding?
Cell spacing is the space between teo cells in a table while padding is the space between the border of a cell and the content in it.
How do you define table borders using CSS?
You use border property to specify table table borders in a CSS file. It defines line weight and if it is a solid or dashed line.
How do you control border widths using CSS?
You use border-width property to define the thickness of borders. You specify the border wdith with :thin/medium/thick
