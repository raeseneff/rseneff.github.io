What is DOCTYPE statement?
A DOCTYPE statement is the basis of the coding to make sure the code follows the specific rules it is supposed to. You used to have to link the rules with it when it was young but now it just needs to be stated.

How do you use: ol; ul; dl?
To use these list types you have to use them in < > so you can tell the document what type of list it is supposed to be making. When you have designated which, for all of them you follow with <"li"> for each point. Each point starts on their own line with the command <"li">. You then close it with of whichever list type you created. Once you run the page, it will show up in that format.

What is the difference between text editor and word processor?
The difference between the two is that text editor lets you add, change, and remove characters from a file while word processor has options for you to format the text or document.

What is the client-server model?
The client-server model is a networking system that shows a relationship between two or more computers. It is a way devices access the information someone stores in servers.

Provide a description and definition for the following network protocols
    FTP
    FTP stands for File Transfer Protocol which is used to transfer filed from one computer to another, it opens two conenction linking the computers communicating with each other.
    HTTP
    HTTP stands for Hypertext Transfer Protocol. It is an application protocol used for collaborative, distributed, hypermedia information systems to allow users to give data to the World Wide Web.
    TCP/IP
    TCP/IP stands for Transmission Control Protocol/Internet Protocol. THis is the labguage a computer uses to access the internet. It has protocols designed to create a network of networks to provide access to the internet.
